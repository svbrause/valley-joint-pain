# Valley Joint Pain Center — Brand Package v1.0

This folder contains everything you need to apply the brand consistently across the website, print materials, and future channels.

## Quick start

| If you want to... | Open this |
|------|-----------|
| Preview every asset visually | `index.html` |
| Read the full brand guidelines | `docs/brand-guidelines.md` |
| Add the logo to the website | `logos/08-horizontal-single-line-primary.svg` (light nav) · `logos/05-horizontal-reversed.svg` (footer) |
| Set up the favicon | `favicons/favicon.svg` |
| Order business cards | `templates/business-card-front.svg` + `business-card-back.svg` |
| Create letterhead | `templates/letterhead.svg` |
| Get a social share image | `templates/og-image.svg` |
| Apply the brand colors | `colors/colors.css` |
| Apply the brand fonts | `typography/typography.css` |

## File formats

All logos are **SVG** — scalable vector graphics that look crisp at any size from a 16-pixel favicon to a wall-sized lobby sign. They open in any modern browser, in Figma/Sketch/Illustrator, and most printers handle them natively.

If you need a PNG or JPG version for a specific application:
1. Open the SVG in a browser
2. Right-click → Save As, or screenshot at desired size
3. Or use a free converter like cloudconvert.com

## What's NOT in this package (but you'll want eventually)

- **PDF versions of business card and letterhead** — your printer will likely want PDFs. Either send them the SVG (most modern printers accept it) or convert via any PDF tool.
- **PNG/JPG renders at standard sizes** — easy to generate on demand from the SVG sources.
- **Animated/video versions of the logo** — for video intros, social reels. Worth a separate project if and when needed.
- **Photography assets** — guidelines are in the brand doc, but actual photos require a photographer.
